# Mari Rodrigues | Cinematic Photography

Official website for photographer Mari Rodrigues, focused on cinematic photography, portrait storytelling, and a clean visual experience.

This project is being developed with scalability in mind, using PHP and MySQL to support future dynamic features while maintaining a minimal and elegant frontend.

---

## Project Overview

The goal of this website is to create a strong online presence for the photographer, highlighting her portfolio through a visually immersive and intuitive interface.

The project focuses on:

- Clean and minimal design
- Emotional and cinematic storytelling
- Easy navigation
- Scalable architecture for future features

Although the website initially behaves as a static experience, it is structured to support dynamic content using PHP and MySQL.

## Application Architecture

The project follows a modular structure to improve maintainability and scalability:

- **Includes system** for reusable components (header, footer, database connection)
- **Global styles** for consistent visual identity
- **Separated assets** (CSS, JS, images)
- **Config-based database connection**, ignored by version control
